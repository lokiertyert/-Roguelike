# Roguelike

Консольная пошаговая игра в жанре Roguelike, сделанная на Python в объектно-ориентированном стиле.

Проект реализован как учебная работа: игрок перемещается по случайно сгенерированной карте, собирает предметы, сражается с врагами, покупает улучшения в магазине и проходит этажи.

## Возможности

- консольная ASCII-игра;
- пошаговое перемещение игрока и врагов;
- случайная генерация карты, врагов и предметов;
- инвентарь с использованием зелий;
- магазин между ходами;
- система урона, опыта, уровня и денег;
- несколько типов врагов;
- победа при прохождении 5 этажей;
- загрузка параметров из JSON-файлов.

## Стек

- Python 3
- стандартная библиотека Python
- `msvcrt` для ввода с клавиатуры

Важно: проект ориентирован на запуск в Windows-консоли, так как используется модуль `msvcrt` и команда `cls`.

## Запуск

1. Убедитесь, что установлен Python 3.
2. Перейдите в папку проекта.
3. Запустите:

```bash
python main.py
```

## Управление

- `W`, `A`, `S`, `D` — перемещение
- `E` — открыть инвентарь
- `Q` — открыть магазин
- `F` — атаковать ближайшего врага
- `Z` — перейти на следующий этаж, если врагов не осталось
- `Esc` — выйти из игры

Примечание: в проекте также поддерживаются русские раскладки клавиш для части действий.

## Цель игры

- пройти все этажи;
- выжить в боях с врагами;
- использовать предметы и магазин для усиления персонажа.

Победа наступает при прохождении 5 этажей.  
Поражение наступает, если здоровье игрока падает до нуля.

## Структура проекта

- `main.py` — точка входа в игру
- `game/` — игровой цикл, боевая логика
- `characters/` — игрок, враги, базовые классы персонажей
- `items/` — предметы и зелья
- `map/` — генерация уровня и игрового поля
- `ui/` — отрисовка интерфейса, инвентарь
- `shop/` — магазин
- `utils/` — загрузка JSON и обработка ввода
- `data/` — игровые настройки, враги и предметы

## Конфигурация

Основные параметры вынесены в JSON-файлы:

- `data/settings.json` — размеры карты, стартовые параметры игрока, число врагов и зелий;
- `data/enemies.json` — характеристики врагов;
- `data/items.json` — параметры предметов и магазина.
